# Suffering Counter

Just a really simple project that counts how much time I have wasted.

It first began as a simple joke, but I slowly started expanding on it, adding test tracking, more data points, etc. Mostly just for fun, but also to learn and experiment with web development.

Also, I really just wanted to use GitHub free hosting xD

---

## About

The project is really simple. Stuff like holidays is currently hardcoded in (might change later).

I originally used an API to fetch Czech holidays, but for some reason it wasn’t working, so I just hardcoded everything instead :]

Tests are fetched from a public Google document because I didn’t want to push a text file every time something changes.

I’m considering switching the holidays to a Google Doc too, or maybe finding a better method because this kinda sucks, but it works.
